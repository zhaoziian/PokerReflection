# PokerReflection

## 软件目的（简要）
PokerReflection 是一个**本地运行**的德州扑克牌局复盘工具，目标是帮助你：
- 把线下手牌过程记录下来（位置、动作、公共牌、show 牌）
- 在牌后做复盘，而不是实时辅助
- 用更直观的方式查看自己的决策习惯和可改进点

> 这个项目偏向“个人成长复盘”，不是纯 GTO 解题器。

---

## 当前进度（简要）
目前已经有一个可运行原型，包含：
- 基础录入：人数、盲注、筹码、Hero 手牌
- 动作录入：按街道记录 Preflop / Flop / Turn / River
- 补充信息：公共牌、show 牌、跑马（最多 3 次）
- 复盘模块：基础 AI 复盘提示、看板与批量审计入口

当前版本已经能用于朋友小范围试用。
